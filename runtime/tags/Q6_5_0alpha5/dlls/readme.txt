Notice! 

The D3DXAS.DLL is only needed when QuArK is compiled with the {$DEFINE QUARK_DIRECT3D}. If not, then the .DLL file can be deleted from the distribution-package.

The QuArkGCF.dll, QuArkSteamFS.dll, QuArkVTF.dll are only needed for QuArKs Source Engine support and loaded when QuArK must acces the game data.
