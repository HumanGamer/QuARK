The texture-browser is used for both the
<a href="intro.mapeditor.overview.html">Map Editor</a> and
<a href="intro.modeleditor.overview.html">Model Editor</a>.<br>
The main differance between the two is that the textures for the
<a href="intro.mapeditor.overview.html">Map Editor</a>
are loaded automatically based on each games configuration location of its game files.<br>
For the <a href="intro.modeleditor.overview.html">Model Editor</a>
skin textures need to be extracted from the game file(s) and
<a href="intro.texturebrowser.tutorials.html#textureimports">imported</a> (most reliable way) or
<a href="intro.texturebrowser.tutorials.html#texturelinks">linked</a> to
for the texture-browser to use and display them.

There are many ways of getting into the texture-browser. The most common way for both
editors is by clicking on its
<a href="intro.mapeditor.menu.html">main menubar</a> item
<a href="intro.mapeditor.menu.html#toolboxesmenu">Toolboxes</a> then click on
<a href="intro.mapeditor.menu.html#toolboxesmenu">Texture Browser...</a>

For the <a href="intro.mapeditor.overview.html">Map Editor</a> another common way is from the
<a href="intro.mapeditor.dataforms.html#polyhedronview">polyhedron-view</a> or
<a href="intro.mapeditor.dataforms.html#faceview">face-view</a> when you are looking at an object
and want to select a texture for it.<br>
See <ref> maped/tutorial/polyhedrons/texturing </ref> for details on how to get to the texture-browser.
