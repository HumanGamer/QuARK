 3.5. Plug-Ins

  [ Prev - Up - Next ] 

Plug-Ins are scripts programmed in Python that modify existing features or add new features to QuArK. They are mainly related to the map editor, which is itself programmed mostly in Python. Plugins as used in QuArK are actually quite different from conventional plugins as found GtkRadiant etc., so here's a bit of background to forestall confusion.

QuArK was originally written entirely in Delphi, but halfway through writing QuArK 5, Armin Rigo discovered Python and decided that it would be better than straight Delphi for programming the QuArK GUI. The architecture he came up with (I don't know what its antecedents are) was that each aspect of the interface would be defined by a plugin file, sitting in the subdirectory plugins/, which would be able to call on both a set of facilities provided by Delphi, and also a collection of facilities provided in Python by files in the quarkpy/.
