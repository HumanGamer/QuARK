 2.1. Tutorial - Creating a map

  [ Prev - Up - Next ] 

This tutorial will guide you through the way QuArK map-editor works, and how you should use it.

Before starting this tutorial, you should check out the 'Configuration of QuArK' section first, to setup QuArK with correct paths to the build-tools and game for texture-images and alike.
