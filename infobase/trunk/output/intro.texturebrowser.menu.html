 Folders menu

 Rowdy - 28 Feb 2001   [ Top ] 

New main folder...  Lets you create a new top level folder. You would then typically create texture lists under this folder, or create subfolders which would then contain texture lists. This folder can be stored in the opened .QRK file (and the textures will only be available for maps also stored in this .QRK file), in a registered add-on (a list of add-ons is presented so you can select one), or in a new add-on. Storing the folder in a new add-on is generally recommended, then the textures will always be available, and you can easily send the add-on to other QuArK users.

New sub-folder  Permits the creation of a sub-folder to contain groups of related textures and 'New texture list' folders. You need to select an existing folder first, then create a new folder under that one. An example of the Dupes texture folder for Quake II. Sub-folders allows you to group your textures very specifically.

Delete folder  Deletes the selected folder, either a main folder or a sub-folder. Note: it is a bad idea to delete the pre-defined folders unless you know what you are doing ;-)

Item Description...  Lets you type in a short description of the folder. This description is displayed next to the folder name.

Reload folders  If you have copied or moved texture files on your disk, Reload folders will refresh the texture display so that it matches whatever your physical file structure is. This is useful when you are regrouping textures into different sub-folders.

Add-ons...  Displays the list of registered add-ons and lets you add or remove them.

Close Toolbox  Closes the texture-browser window.


 Edit menu

 Rowdy - 28 Feb 2001   [ Top ] 

Undo  Allows you to quickly and simply undo the last change.

Undo/Redo  Presents the QuArK Undo/Redo window so you can select a number of changes to undo or redo.

Cut  Removes the selected item (folder, texture etc.) and places it into the clipboard.

Copy  Copies the selected item (folder, texture etc.) and places it into the clipboard.

Paste  Copies something from the clipboard into the texture-browser.

Delete  Deletes the selected item (folder, texture etc.).

Copy object as  Copies the selected item (folder, texture etc.) and places it into the clipboard, but allows you to select from a second level menu exactly what type of object you are copying.

Paste special...  Prompts you to select what type of object you are pasting, then copies it from the clipboard into the texture-browser. The Paste-special window is very similar to the Object-properties window.

Import files  This will prompt you to select one or more files, and then display them in the current folder. If you link to the files, then only a reference to the files is placed into the add-on, but if you move the actual files later on they will no longer be available in the texture-browser. If you copy the files into the add-on, they will always be available, even if you move or delete the original files, but your add-on will be a lot bigger.

Open selection...  This is the default action if you double-click somewhere, and opens a folder, texture, or selects a texture for use in your map.

External editor  Opens the external editor associated with the selected object. External editors are configured in 'External editors'.

Object properties...  Opens a window displaying the object's properties, such as the type and size. Also allows you to copy the item to the clipboard.

Configuration...  Runs the 'Configuration of QuArK'.


 Object properties

 Rowdy - 28 Feb 2001   [ Top ] 

Displays some properties about the selected object, from the Object properties...  or Paste special...  menu options.

The window shows the object pathname, the file type and size, and then a list of types the object can be converted to, and a button to copy the object to the clipboard or paste depending on where the window was displayed from.

The Save as...  button allows you to save the object as an external disk file. Useful if you want to play with the file outside the QuArK environment.


 Toolboxes menu

 Rowdy - 28 Feb 2001   [ Top ] 

New files...  Displays the New files toolbox.

New map items...  Displays the New map items toolbox.

Texture Browser...  Displays the texture-browser toolbox, except that you are already here ;-)

Main window  Display the main QuArK window.

New map  This refers to any maps which you are currently editing. There may be one or more entries here.

Minimize window  Minimises the texture-browser window.


 New texture icons

 Rowdy - 28 Feb 2001   [ Top ] 

These buttons, one for each game, allow you to create a new blank texture. Once you click one of these, the 'Texture details' view will appear, allowing you to edit and change the properties of the texture.


 New texture list

 Rowdy - 28 Feb 2001   [ Top ] 

This creates a new folder which can store a list of textures, and which also shows you a preview of the textures when you select the group. Once you have added a new texture-list, click once on the name to rename it.


 Link to folder

 Rowdy - 28 Feb 2001   [ Top ] 

If you have a large collection of textures you can use this to create all the groups for you. Create a new main folder (see above), then click the Link to folder  button, you will be asked to select the directory containing the textures. After a short search QuArK will add one 'New texture list' to the new main folder for each .PAK or .WAD file that contains textures and each folder under the one you select which also contains textures.
