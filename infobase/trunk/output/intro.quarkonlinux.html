 Compiling a Map under Linux

 slightcrazed, Peter Brett - 15 Jan 2004   [ Top ] 

For one reason or another when you run '[GAME] -> GO! (textures+build+[GAME])' it will only run the first build tool. So you may have to run the BSP, VIS, and RAD portions manually.

It may be necessary to create a shell script to do the compile process externally to QuArK, and then reconfigure the menu to point to it. Please see your build tools' documentation and the man/info pages for your shell for more information.
