title: Registering at SourceForge
author: Armin Rigo

Connect to the <a href="http://sourceforge.net">SourceForge</a>
and register yourself as a new user. Then contact a project administrator (currenlty
arigo@<!-- -->planetquake.com or Alexander@<!-- -->users.sourceforge.net) and give your
login name so that we can add you to the quark project members list.

Follow the instructions given in the file
<a href="http://asp.planetquake.com/dl/dl.asp?quark/source/ssh-pack.zip">ssh-pack.zip</a>
to configure WinCVS for non-anonymous access to the repository.
